This plugin lets you show your reviews from klantenvertellen.nl on your website with shortcodes.

== Description ==

Nederlands
Deze plugin laat je beoordelingen zien van klantenvertellen.nl rechtsstreeks op je eigen website met shortcodes op een makkelijke en zelf instelbare manier.

Er zijn op dit moment drie verschillende shortcodes om je beoordelingen te tonen
- Toon algemene statistieken
- Toon beoordelingen van klanten
- Toon beoordelingen naast elkaar tot 3 beoordelingen

English
This plugin will let you show your klantenvertellen.nl reviews on your website with shortcodes in an easy and extremely customizable way.

Currently there are three different shortcodes to show your reviews
- Display the overal statistics
- Display reviews from customers
- Display reviews side by side up to 3 reviews

== Installation ==

Nederlands
Het installeren van de plugin op je Wordpress website.

1. Installeer automatisch door middel van het `Plugins` menu en de `Nieuwe plugin` knop (of upload de hele `reviews-for-kv` map naar de `/wp-content/plugins/` map)
2. Activeer de plugin
3. Gaar naar `Klantenvertellen` => `Instellingen`
4. Voer hier de URL in naar je gepersonaliseerde XML link van klantenvertellen.nl
5. Druk op de `Indexeer velden` knop
6. Ga naar `Klantenvertellen` => `Configurator`
7. Verander alle instellingen en de HTML template naar je eigen wens
8. Voeg de shortcodes toe aan de pagina.

English
Installing the plugin on your Wordpress website.

1. Install automatically through the `Plugins` menu and `Add New` button (or upload the entire `klantenvertellen-reviews` folder to the `/wp-content/plugins/` directory)
2. Activate the plugin
3. Go to `Klantenvertellen` => `Settings`
4. Enter the URL to your personalised XML file from klantenvertellen.nl
5. Hit the `Index fields` button
6. Go to `Klantenvertellen` => `Configurator`
7. Change any settings or customize the HTML for the shortcodes
8. Insert the shortcodes any way you like into a page.
